Testing

To test these, install the testing bookmarklet of your choice in your browser's bookmarks or favorites bar. See the Test Cases for example markup. This markup includes the expected results you should see based on the source code for that test case. Activate a test bookmarklet on any test page and check that the tooltip values match the expected results.

This open source project welcomes all interested collaborators and contributions. If you have a bug or a feature suggestion, please create an issue in GitHub. If you want to contribute a code fix or new feature, feel free to fork the bookmarket repository in GitHub and submit a pull request.
